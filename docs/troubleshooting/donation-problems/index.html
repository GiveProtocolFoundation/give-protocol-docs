Common Issues

Donation Not Showing

  • Allow time for blockchain confirmation
  • Check transaction hash on block explorer
  • Verify the receiving organization

Wrong Amount Donated

  • Double-check before confirming transactions
  • Contact support if there’s a discrepancy

Receipt Not Received

  • Check your spam folder
  • Download from your dashboard
  • Contact the organization
